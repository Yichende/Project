# 智能 BI

该代码仓库为前端部分仓库。

技术栈：Ant Design Pro+ Umi4 + React + TypeScript + Echarts +Ant Design 组件库

该项目仍在测试改进中，存在部分调试用代码，仅供参考。

### 功能

1. 通过输入分析需求自动生成 **图表** 以及 **分析结果**。
2. 搜索已生成的图表。
3. 删除图表。
4. 异步生成图表，生成过程中可以跳转至其他页面。

> 注：图表由 Echarts 中 options 生成，由 AI 生成 options 部分代码，会出现图表参差不齐的现象。
